# 🛒 MERN E-Commerce App

A full-stack e-commerce web application built using the MERN stack.

## 🚀 Features

- User Authentication (JWT)
- Product Listing
- Add to Cart (Persistent)
- Checkout System
- Order Management
- Protected Routes

## 🛠️ Tech Stack

- Frontend: React, Tailwind CSS
- Backend: Node.js, Express
- Database: MongoDB
- Auth: JWT

## 📦 Installation

``bash
git clone <your-repo>
cd mern-ecommerce
## BackEnd
cd backend
npm install
npm run dev
## FrontEnd
cd frontend
npm install
npm run dev
